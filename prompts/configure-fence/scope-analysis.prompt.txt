Decision guidance:
- session:

  - experimental changes,
  - changes that should not affect future sessions.
- workspace:
  - repository-specific policy,
  - team/project-local defaults,
  - behavior intended for this working tree only.

  - user-wide defaults,
  - behavior intended across repositories.

  - request is ambiguous,

- session overrides workspace/global,
- workspace overrides global,
- only one scope file is selected at launch.
